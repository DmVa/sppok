Db Updater. Tool for detect file changes and update MS SQL databases from script files.
Prerequisites: MS SQL Express, Windows 7/8/10/11.
Usage scenario:
Run application. Define your server connection string.
Define databases to update. At least one database should be defined.
Define local folders which will be scanned.
Click 'Scan changes', At first run system will recognize all files as new. Click 'All Register without run'.
Change local files, like get new version from source control. Click 'Scan changes', system will show changed files.
Select changed file and click 'Run script from file'.
