Db Updater. Tool for detect file changes and update ms sql database from script files
Prerequisites: MS Sql Express for internal use, Windows 7/8/10/11.
Usage scenario:
Run application. Define server connection string.
Define databases to update. At least one database should be defined.
Define local folders which need to scan file changes.
Click scan changes, for first run system will recognize all files as new. Click 'All Register without
Change local files, like get new version from source control. Click scan changes, system will show changes files.
Select changed file and click 'Run script from file'.
